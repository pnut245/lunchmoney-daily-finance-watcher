# One Number Today Design

The number is the interface.

## Daily Screen

- White background for positive or zero.
- Black oversized number for positive or zero.
- Red background only when negative.
- White oversized number only when negative.
- No dollar sign in the hero number.
- No cards.
- No charts.
- No graphs.
- No progress bars.
- No warning badges.
- No safe/caution/danger labels.
- No green, yellow, orange, or multi-state color system in V1.
- No object/category pill, Week, Dopamine, or dashboard metadata in the hero.

The number may be nearly cropped. It should feel closer to a Swiss poster or Apple ad than a finance app.

Low positive numbers stay black on white. Only values below zero switch to red.

## Widget And Watch

The widget, watch, and lockscreen should show only the number.

Positive or zero uses black on white. Negative uses white on red. No category detail, week/month context, dashboard layout, object image, state label, or dopamine amount belongs in the V1 widget.

## Settings And Vault

The web settings surface can show controls, but it should remain calm and sparse. Vault rows are for long-term reflection, not daily anxiety.
